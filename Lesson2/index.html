Cascading Style Sheets - Каскадные таблицы стилей

Свойства обращенные через селектор по тегу коснуться всех элементов этой группы

Если мы хотим стилизовать конкретный элемент - мы используем селектор по классу

CSS можно подключить 3-мя способами

  • Первый (приорететный) - создать файл со стилями СSS
  • Второй - внутренний CSS через тег "Style"
  • Третий - инлайновый СSS, указываем стили внутри конкретного тега
