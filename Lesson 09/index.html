Lesson 09. DOM into

DOM- Document Object Model

все элементы на страницы можно представить в виде глобального обьекта document

что бы находить элементы в DOMе нужно добавить что-то уникальное(какую то отличительную особенность (класс или ID))

  • по тегам - document.getElementsByTagName
  • по классам - document.getElementsByClassName
  • по ID - document.getElementById
  • querySelector - document.querySelector - универсальный метод поиска, класс уточняется через точку, ID- уточняется через #. При этом в ответ прийдет только один ответ, первый нашедшийся
